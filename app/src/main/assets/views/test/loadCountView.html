Hot load and load count

Home

This page has been requested {{LoadCount}} times.

The count above increments each time the Controller is called to render this page. This is the equivalent of your app doing some kind of 'action' or performing a request.

If you go back to the Home page and visit this page again, the number "{{LoadCount}}" should increase.

However, if you open the file at assets/views/test/loadCountView.html and edit it (causing a hot re-load) then the number should not increase. This demonstrates that the controller was not called again, and the existing page was simply re-loaded.

Here is a big pile of Emoji. Some are more recent and might not display on very old phones. Use these rather than custom graphics if you want a smaller APK, or are lazy (or both).

Emojipedia has a reference page for Android here: https://emojipedia.org/google/

There is an excellent tool "BabelMap" from https://babelstone.co.uk/Software/BabelMap.html which can help you navigate the maze of Unicode glyphs.
